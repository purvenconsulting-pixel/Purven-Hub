# Utility Tools Build Prompt - Phase 2

## Build the following 5 professional tools for Purvenhub.com:

### Tool 6: AI Resume Bullet Point Generator
- Input: Job title, core achievement/task
- Action: Convert raw tasks into high-impact, results-oriented bullet points using action verbs and metrics.
- Output: 3-5 optimized bullet points.

### Tool 7: AI LinkedIn Headline Generator
- Input: Job title, top 3 skills, years of experience
- Action: Generate 5 variations of compelling LinkedIn headlines (Professional, Creative, Achievement-focused).
- SEO: Target "LinkedIn headline examples", "best LinkedIn headlines".

### Tool 8: AI YouTube Title & Description Generator
- Input: Video topic, target audience, 3 keywords
- Action: Generate 5 click-worthy titles and a comprehensive, SEO-optimized video description.
- Features: Include emoji use, timestamp placeholders, and social link placeholders.

### Tool 9: AI Tweet/Thread Generator
- Input: Main hook or topic
- Action: Generate a single viral tweet or a 5-tweet thread with structured storytelling (Hook → Value → Call to Action).
- Vibe: Engaging, punchy, Twitter-native.

### Tool 10: AI Product Description Generator
- Input: Product name, 3 key features, target audience
- Action: Write a persuasive, benefit-driven product description (200 words max).
- Style: E-commerce standard (Benefit-Feature-Benefit).

## General Design Spec:
- Integration with Purvenhub.com existing UI
- Dark/light mode support
- Copy to clipboard
- Mobile responsive
- Fast load time
